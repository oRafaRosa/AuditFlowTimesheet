# AGENTS.md

## Git Workflow
- Depois que uma alteracao passar na validacao, sempre criar commit.
- Depois do commit, sempre executar `git push` para `origin main`.
- So deixar de commitar ou fazer push se o usuario pedir explicitamente.

## Comentarios No Codigo
- Sempre escrever comentarios em pt-BR.
- Usar linguagem simples, direta e natural, como se o proprio Rafael tivesse escrito.
- Evitar tom formal, academico ou com cara de texto gerado por IA.
- So adicionar comentario quando ele realmente ajudar a entender o trecho.
